¡Novedades! - Versión 1.0.2 (20/02/2020)

Sectorización

Ahora el sistema permite realizar un ABM de sectores y asignarlos a los usuarios.

Referentes

Los solicitantes pueden marcarse como referentes.

FIX

  • Ahora el cursor del mouse cambia cuando se para encima de un ticket en una lista.
  • Se unificaron los colores de los pop-ups.

Acceso externo

A partir de ahora, el sistema permite acceder de forma remota desde cualquier dispositivo utilizando el link http://190.220.159.37:3977/.

En celulares se recomienda usar la aplicación de forma horizontal.

Acceso desde el PIUA

A partir de ahora, el sistema se encuentra disponible para acceder en el PIUA.

Sectorización

  • Los usuarios ahora tienen identificado a que sector pertenecen.
  • Los referentes tienen la función de visualizar los tickets enviados por su sector.

Reapertura de tickets

Una vez finalizado el ticket, el solicitante podrá volver el estado a pendiente si considera que el problema no se encuentra resuelto.

FIX

  • Los usuarios que hayan sido copiados en otros tickets ahora podrán visualizarlo correctamente en su pantalla
  • En las listas de tickets, ahora se recortará el asunto para dar mas espacio a otra información. Si el usuario ubica el mouse encima del asunto, éste desplegará el texto completo.
